üì¶ What are Variables?

Think of variables like labeled boxes!
You can put things inside them and use them later.
// We put a number in a box called "size" let size = 50; // We put a color in a box called "color" let color = "blue"; // Now we can use them to draw! drawCircle(size, color);

üîÅ What are Loops?

Loops help us repeat actions!
Instead of writing the same code 10 times, we can use a loop!
// Draw 10 circles in a row for (let i = 0; i < 10; i++) { drawCircle(i * 50, 200); } // The loop does this automatically: // i=0: draw at position 0 // i=1: draw at position 50 // i=2: draw at position 100 // ... and so on!

üéØ Try This!

Change the number of circles and watch the loop work its magic!

ü§î What are If/Then Statements?

If/Then helps us make decisions!
"IF something is true, THEN do this. ELSE do that."
// Change color based on size if (size > 80) { color = "red"; // Big circles are red } else if (size > 40) { color = "yellow"; // Medium circles are yellow } else { color = "blue"; // Small circles are blue }

üéÅ What are Functions?

Functions are like recipes!
You write the steps once, then use them over and over!
// Create a function to draw a smiley face function drawSmiley(x, y, size) { // Draw the face drawCircle(x, y, size, "yellow"); // Draw the eyes drawCircle(x - 15, y - 15, 8, "black"); drawCircle(x + 15, y - 15, 8, "black"); // Draw the smile drawSmile(x, y); } // Now we can draw lots of smileys easily! drawSmiley(150, 150, 80); drawSmiley(350, 150, 60);

üéØ Try This!

Each smiley uses the SAME function, but with different values! That's the power of functions!
