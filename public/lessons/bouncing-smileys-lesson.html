üë• Many Smileys Moving!

Using loops to make many smileys!
Each smiley has its own position and speed.
They all move at the same time!
// Array to store many smileys let smileys = []; // Create 5 smileys with different speeds for (let i = 0; i < 5; i++) { smileys.push({ x: 100 + i * 50, y: 100, speedX: 2 + i, speedY: 3 - i }); } // Move each smiley for (let smiley of smileys) { smiley.x += smiley.speedX; smiley.y += smiley.speedY; }
