üîÅ Lesson 2: Loops Make it Easy!

Loops repeat code automatically!
Instead of writing the same line 10 times, use a for loop. This is one of Python's most powerful features!

For Loops in Python:

for i in range(4):
    forward(100)
    right(90)
# This draws a square with just 4 lines!

for i in range(360):
    forward(1)
    right(1)
# This draws a circle!

Functions let you reuse code!
Define a function once, then call it many times. You can even pass different values to make it flexible!

Defining Functions:

    for i in range(4):
        forward(size)
        right(90)
